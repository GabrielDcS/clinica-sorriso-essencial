# Clínica Sorriso Essencial

Landing page demonstrativa para uma clínica odontológica de Salvador do Sul — RS, criada para gerar avaliações e contatos pelo WhatsApp.

## Tecnologias

- Next.js 16
- React 19
- TypeScript
- CSS responsivo

## Desenvolvimento

```bash
npm install
npm run dev
```

Acesse `http://localhost:3000`.

## Produção

```bash
npm run build
npm start
```

Os nomes, depoimentos, telefone, endereço e demais dados comerciais são fictícios e devem ser substituídos antes do uso real.
